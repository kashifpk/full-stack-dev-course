# Full-Stack Development Course: Python, FastAPI, TypeScript & Vue.js

A self-paced course designed for experienced developers returning to the field. Build a complete **Job Board Application** while learning modern full-stack development.

## Target Audience

- CS background with previous development experience (PHP, Python, SQL)
- 5+ years away from active development
- Targeting fullstack or backend-heavy roles

## What You'll Build

A fully functional **Job Board Application** with:
- Job listings with search and filtering
- User roles: Job Seekers, Employers, Admins
- Resume uploads and application tracking
- Authentication with JWT
- Responsive UI with dark/light themes
- Containerized deployment

## Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Python 3.11+, FastAPI, SQLAlchemy 2.0, Alembic |
| Database | PostgreSQL 16 |
| Frontend | Vue 3 (Composition API), TypeScript, Pinia, Vue Router |
| UI | PrimeVue 4, Vanilla CSS |
| Testing | pytest, Vitest, Vue Test Utils |
| DevOps | Docker/Podman, Docker Compose |

## Time Estimate

At 2-3 hours/day:
- Each module: 3-5 days
- Total course: 8-12 weeks

## How to Use This Course

1. **Read** the module content
2. **Complete** the exercises within each section
3. **Build** the assignment (adds to the Job Board project)
4. **Take** the quiz to verify understanding
5. **Move on** when comfortable (no rush!)

## Prerequisites

- Basic programming knowledge
- Familiarity with SQL concepts
- Command line basics
- Git basics (we'll use it throughout)
